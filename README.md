Directory for the data download and analysis code relating to the referenced PACE hurricane-response paper. 
Included is a jupyter notebook which retrieves PACE products and other supporting data for the 4 chosen hurricanes. Data are organized and processed, and publication-quality figures are made.

Note: running this code will download ~200GB of data to a .gitignored folder "local" in this repository. Figures are written to a new folder as well
